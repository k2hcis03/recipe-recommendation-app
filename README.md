# 목적 기반 레시피 추천 앱

이 프로젝트는 사용자의 목적(다이어트, 건강, 기념일 등)에 따라 적절한 레시피를 추천해주는 웹 애플리케이션입니다.

## 기능

- 목적별 레시피 추천 (다이어트, 건강, 기념일)
- 레시피 상세 정보 제공 (재료, 조리방법)
- 이미지와 함께 직관적인 레시피 표시

## 기술 스택

- Backend: FastAPI
- Frontend: HTML, Tailwind CSS
- Database: SQLite
- Template Engine: Jinja2
